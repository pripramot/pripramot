ปัญหาการเชื่อมต่อในหน้าต่างเอาต์พุต Duet AI
หากคุณเห็นข้อผิดพลาดในการเชื่อมต่อหรือปัญหาการเชื่อมต่ออื่นๆ ในหน้าต่างเอาต์พุต Duet AI ให้ลองทำดังนี้:

กำหนด ค่าไฟร์วอลล์ของคุณเพื่ออนุญาตการเข้าถึงoauth2.googleapis.comและ cloudaicompanion.googleapis.com

กำหนดค่าไฟร์วอลล์ของคุณเพื่ออนุญาตการสื่อสารผ่าน HTTP/2 ที่ gRPC ใช้

คุณสามารถใช้grpc-health-probeเครื่องมือเพื่อทดสอบการเชื่อมต่อได้ การตรวจสอบที่สำเร็จจะให้ผลลัพธ์ต่อไปนี้:

$ grpc-health-probe -addr cloudaicompanion.googleapis.com:443 -tls error: this server does not implement the grpc health protocol (grpc.health.v1.Health): GRPC target method can't be resolved

การตรวจสอบที่ไม่สำเร็จส่งผลให้เกิดผลลัพธ์ต่อไปนี้:

timeout: failed to connect service "cloudaicompanion.googleapis.com:443" within 1s

หากต้องการดูรายละเอียดเพิ่มเติม ให้รันสิ่งต่อไปนี้ก่อนgrpc-health-probe:


export GRPC_GO_LOG_SEVERITY_LEVEL=info
