# GSTORE 🛡️🦀

**GSTORE** (GTS Alpha Store & Forensics) คือโปรแกรมค้นหาไฟล์ประสิทธิภาพสูงและระบบจัดการแอปพลิเคชันยุคใหม่ สร้างขึ้นด้วยเทคโนโลยี **Tauri v2 (Rust + React)** เพื่อความเร็ว แรง และเบาที่สุด

## ✨ Key Features
- **High-speed Search Engine:** สแกนและค้นหาไฟล์ด้วยความเร็วระดับ Rust
- **Auto-Sync Manifest:** ระบบ Store ที่อัปเดตรายการแอปผ่าน GitHub อัตโนมัติ
- **Forensic Extraction:** วิเคราะห์ Metadata และ Signatures ของไฟล์โดยตรง (แงะ!)
- **AI-Powered Discovery:** ค้นหาเครื่องมือใหม่ๆ ด้วยพลังของ Exa AI

## 🚀 Getting Started
1. `cd frontend && npm install`
2. `cd src-tauri && cargo fetch` (หากมี Rust)
3. `npm run tauri dev`

## 🛡️ Forensic Logging
ทุกการทำงานจะถูกบันทึกไว้ในโฟลเดอร์ `.gstore/logs/` เพื่อการตรวจสอบย้อนหลัง
